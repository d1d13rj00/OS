# miniOS

## General idea

The goal of this project is to to create a very simple operating system, demonstratng the basics of how an operating system works.

### Features of the OS

- Interactivity
 - interactive shell
- Basic applications
 - echo : outputs data
 - read : read input data
 - run : run a program
- System calls
 - output data
 - read input data
 - load program
 - execute program
 - reset
 
### Implementation

This will be dependant on CPU architecture chosen.
